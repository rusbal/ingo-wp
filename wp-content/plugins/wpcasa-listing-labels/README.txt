Feature specific listings by visually highlighting them with a label.

== Description ==

Please notice that this plugin is an add-on for [WPCasa](https://wpcasa.com) and will NOT work without the core plugin.

* Website: [wpcasa.com/downloads/wpcasa-listing-labels](https://wpcasa.com/downloads/wpcasa-listing-labels)
* Documentation: [docs.wpsight.com](http://docs.wpsight.com)

== Installation ==

Download the plugin ZIP file from your WPCasa account to your local computer. Then go to _WP-Admin > Plugins > Add New > Upload Plugin_ and browse to the plugin on your local computer. Finally activate it.

For more information about how to get started please [read our documentation](http://docs.wpsight.com).
